# OpenCall Token — 艺术机会源站调研报告（含网址）

**版本:** v1.1
**日期:** 2026-07-01
**用途:** 为 OpenCall Token 爬虫与数据聚合提供源站清单（新增网址列）

---

## 说明

本文档系统梳理了全球范围内发布艺术公开征集（Open Call）、驻留项目（Residency）
和艺术家资助（Grant）信息的主要渠道，包括网站、社群、RSS/邮件列表等。
每个渠道标注了权重与网址，用于指导后续爬虫优先级和数据聚合策略。

**权重含义：**

- ★★★★★ (5): 核心数据源，必须覆盖，覆盖最广/质量最高
- ★★★★ (4): 重要数据源，强烈推荐覆盖
- ★★★ (3): 补充数据源，有一定价值
- ★★ (2): 窄众渠道，特定场景有用
- ★ (1): 边缘渠道，数据量少或偶发
