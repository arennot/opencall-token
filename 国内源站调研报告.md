# OpenCall Token — 国内艺术机会源站调研报告

**版本:** v1.0
**日期:** 2026-07-01
**用途:** 为 OpenCall Token 爬虫与数据聚合提供国内源站清单

---

## 说明

本文档系统梳理了中国国内（含港澳台）发布艺术公开征集（Open Call）、驻留项目（Residency）
和艺术家资助（Grant）信息的主要渠道，包括网站、微信公众号、小红书、豆瓣、微信群等。
每个渠道标注了权重，用于指导后续爬虫与数据聚合策略。

**权重含义：**

- ★★★★★ (5): 核心数据源，必须覆盖
- ★★★★☆ (4): 重要数据源，强烈推荐覆盖
- ★★★☆☆ (3): 补充数据源，有一定价值
- ★★☆☆☆ (2): 窄众渠道，特定场景有用
- ★☆☆☆☆ (1): 边缘渠道，数据量少或偶发

**注意：** 国内艺术机会发布渠道高度分散，大量信息通过微信生态（公众号/群/朋友圈）传播，
结构化程度低，人工 + 半自动化采集是更可行的方式。

## 六、采集优先级策略

### 第一批（P0 -- 核心数据源，立即覆盖）

1. 雅昌艺术网 -- 国内最大艺术门户（网站+公众号双覆盖）
2. Hi艺术 / 艺术头条 -- 行业媒体，高频发布
3. 国家艺术基金 cnaf.cn -- 政府最高级别资助
4. 中国美术家协会 -- 官方征集主渠道
5. 公众号：艺术新闻中文版 / 艺术中国

### 第二批（P1 -- 重要补充）

6. PSA / UCCA / 红砖 / A4 等机构官网 -- 驻留+展览征集
7. 豆瓣相关小组 -- 豆瓣爬虫采集
8. 小红书话题标签聚合
9. 各省市文化艺术基金官网 -- 地方资助信息
10. 新星星艺术奖 / 三影堂摄影奖 / 华宇青年奖 等竞赛官网

### 第三批（P2 -- 广度覆盖）

11. 微博官方账号监控 -- 机构号和媒体号
12. 地方美协/文联官网 -- 各省市分散信息
13. 更多机构公众号逐个接入
14. 各大学/美院官网 -- 高校艺术驻地计划

### 微信群策略（特殊处理）

- 微信群是信息密度最高但最难以技术手段采集的渠道
- 建议策略：人工嵌入 3-5 个核心群 + 群内志愿者转发至收集机器人
- 或与群主合作，允许 API 接口接入群消息

## 附录：国内艺术生态关键特征总结

| 维度 | 特征 | 对采集策略的影响 |
|------|------|----------------|
| 发布渠道 | 高度分散，微信公众号是第一信息流 | 需要公众号采集能力 |
| 结构化程度 | 低，多数通过推文/群聊传播 | 半自动+人工辅助为主 |
| 信息时效性 | 短，征集信息发布后即开始倒计时 | 需及时采集（日内） |
| 竞争格局 | 国内缺少一站式聚合平台 | 这是 OpenCall Token 的机会 |
